Travel Mate

Tech Stack: JavaScript, React, HTML, CSS, exchange-api

A web application that allows users to plan, save, and manage travel itineraries. Includes ablitity to log trips and associated activities. Allows users to track packing lists and important travel reminders. Incorporates a currency exchange API to support international travel planning.
